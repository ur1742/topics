# OpenAlex Research Database Web Application

Веб-приложение для работы с базой данных научных публикаций OpenAlex.

## Функционал

1. **Поиск и скачивание записей** - Поиск работ в базе OpenAlex по ключевым словам, фильтрация и скачивание результатов в JSON/CSV форматах
2. **Анализ статистики** - Визуализация распределения публикаций по годам, тематикам, цитированиям и другим метрикам
3. **Классификация тематик** - Обучение нейросетевого классификатора на найденных работах и определение тематики для новых текстов

## Установка

```bash
pip install -r requirements.txt
```

## Запуск

```bash
python app.py
```

Приложение будет доступно по адресу: http://localhost:5000

## Использование

1. Введите поисковый запрос (например: "machine learning", "artificial intelligence")
2. Укажите количество результатов (до 100)
3. Нажмите "Найти" для поиска в базе OpenAlex
4. Просмотрите статистику найденных работ
5. При необходимости обучите нейросеть на найденных данных
6. Используйте классификатор для определения тематики новых текстов
7. Скачайте результаты в JSON или CSV формате

## API Endpoints

- `POST /api/search` - Поиск работ в OpenAlex
- `GET /api/statistics` - Получение статистики по результатам поиска
- `POST /api/train` - Обучение модели классификации
- `POST /api/classify` - Классификация текста
- `GET /api/download?format=json|csv` - Скачивание данных

## Технологии

- **Backend**: Flask, requests, pandas, numpy, scikit-learn
- **Frontend**: Bootstrap 5, Chart.js
- **API**: OpenAlex API (https://api.openalex.org)
